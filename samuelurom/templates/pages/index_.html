Featured Projects

CLI Inventory Manager

Python | Open Source

This is a command-line based application that allows you to manage your inventory and keep track of your products. With this program, you can add new products to your inventory, restock existing products, search and view your inventory in a variety of ways. The application is built with strong OOP concepts and the data is persisted in an external CSV file. View this project on my GitHub.

Simple Task Management System

Python | Open Source

This task management system is a simple CLI tool. It allows users to create, view, and manage tasks from the terminal. Only the admin is authorized to register new users, add a new task and assign to a valid user, edit uncompleted tasks, and generate system reports. Other users can easily enter commands to view all tasks, view and mark tasks assigned to them as completed. The system stores user and tasks data in external text files. View this project on my GitHub.

Rock Paper Scissors - The Game

Vanilla JavaScript, HTML, CSS | Open Source

Rock Paper Scissors is a classic hand game that has been enjoyed by people of all ages for generations. I have built this version using JavaScript and created a beautiful user interface to enhance the gameplay experience. To play the game, choose between rock, paper, or scissors by clicking on one of the buttons on the screen. The computer will also make a selection and the winner will be determined based on the game rules. As you play, your score will be tracked and displayed on the screen. Can you outsmart the computer? Give it a try and find out! View this project on my GitHub.
